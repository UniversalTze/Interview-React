# ReadySetHire
A React project that helps with interview and the process of conducting it. Front-end components done with Bootstrap. 

## Author
Tze Kheng Goh

## Instructions
- Git clone the repo. 
- After git cloning navigate to root and run `npm install`
- run `npm run dev` 

You should have locally hosted version of the website. Enjoy!
